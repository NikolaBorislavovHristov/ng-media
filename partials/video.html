html5-video

The html5-video directive enables the creation of data-bound videos, with support for overlaid HTML and custom controls.

The following example shows a simple, inline utilization of the directive without any scope-bound expressions.

      <video data-html5-video="['http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4',
                                'http://clips.vorwaerts-gmbh.de/big_buck_bunny.webm',
                                'http://clips.vorwaerts-gmbh.de/big_buck_bunny.ogv']"
             data-tracks="{
               src: 'captions/bunny-en.vtt',
               type: 'text/vtt',
               kind: 'subtitles',
               srclang: 'en-US'
             }"
             data-preload="true"
             data-autoplay="true"
             data-controls="true">
      </video>

The video sources and text tracks may be specified using both the `src` attribute, and the `html5-video` attribute (including variations on the attribute names based on the AngularJS Directive normalization rules).

Additionally, it is possible to specify sources/tracks in a number of ways:

  • String literals (`data-html5-video="http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4"`)
  • Interpolated strings (`data-track="./assets/captions/{{video}}-{{locale}}.vtt"`) (Warning: AngularJS will not allow complex interpolation expressions for the attributes `href` and `src`)
  • Variables in $scope (`data-src="videos[$index]"`)

Many other attributes are also supported. API reference coming soon!
