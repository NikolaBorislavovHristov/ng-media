html5-video overlay

The html5-video directive provides a simple mechanism for HTML overlays

This example shows a simple use of the directive with an inline HTML overlay.

The overlay may contain AngularJS forms which live within the same application, as no iframe is used.

      <video data-html5-video="['http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4',
                                'http://clips.vorwaerts-gmbh.de/big_buck_bunny.webm',
                                'http://clips.vorwaerts-gmbh.de/big_buck_bunny.ogv']"
             data-track="{
               src: 'assets/captions/bunny-en.vtt',
               type: 'text/vtt',
               kind: 'subtitles',
               srclang: 'en-US',
               label: 'Subtitles'
             }"
             data-controls="true"
             data-width="640"
             data-height="360">

        <!-- Overlay code -->
        <p class="shadow">
          It's super easy to overlay HTML over HTML5 media widgest using ng-media!
        </p>
        <img src="/assets/img/anigif.gif" title="Wow!" width="200" />
      </video>

While this example showcases a simple inline overlay, it is also possible to overlay a template URL using the `overlay-url` attribute.

This could be useful for different things, like adding "remixing" controls similar to the WebMaker project, or providing an authentication form, or even just a basic watermark for your webapp.

You might have lots of different uses for it, and it's easy to accomplish
