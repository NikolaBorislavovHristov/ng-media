This concludes the tour of ng-media, for the time being. More demonstrations should be available as more functionality is added to the module!

Though the list of features is not at all exhaustive at this time, it is hoped that this can be cleanly expanded on. Here are some of the things we'd like to support:

  • Playlists
  • Improved preloading support
  • Audio tag support
  • WebAudio support
  • WebRTC support
  • Support for media supplied by browser plugins
  • Better API documentation

I hope you find this interesting and useful, and I look forward to hearing your comments.
